BaseView

(
  • [p_data=null]
  • [p_CSSClassName=null]
)

Defined in ../../source/coffee/slikland/navigation/display/BaseView.coffee:96

Parameters:

  • [p_data=null] Object optional

    Data object sets the default and/or custom values of properties of view for navigation controller.
    If this object it's not null, some default properties are not required explained below:

    Default Key Type Required
    id String Yes
    class String Yes
    route String / RegExp No
    content String / JSON No
    cache Boolean No
    parentView String No
    destroyable Boolean No
    loadContent Boolean No
    snap (only for scroll navigation type) Boolean No
    subviewsWrapper Selectors No
    attachToParentWrapper Selectors No
  • [p_CSSClassName=null] String optional

Example:

{
    "id":"home",
    "class":"template-home-view", //the valid formats are "ClassName", "class-name", "class name" or "class_name"
    "route":"/", //valid formats are String or RegExp
    "content":"data/home.json",
    "cache":true,
    "parentView":"someViewID", //the unique ID of parent view
    "destroyable":true,
    "loadContent":true,
    "snap":true, //only for scroll navigation type
    "subviewsWrapper":"CSSSelector", //like #ID or .className etc
    "attachToParentWrapper":"CSSSelector" //like #ID or .className etc
}
